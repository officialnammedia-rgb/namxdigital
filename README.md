# NamxDigital - Women's Day Surprise 💖

A beautiful interactive Women's Day surprise website with balloon popping animations!

## 🎀 How It Works

When someone visits the website, they see:
1. **Opening Screen** - Shows the "Queen Photo" with a crown and "You're a Queen" message
2. **Balloon Screen** - 8 colorful balloons to pop
3. **Photo Reveals** - Each balloon reveals a photo with a romantic message
4. **Final Screen** - A heartfelt "Happy Women's Day" celebration

## 📷 How to Add Your Photos

Simply replace the placeholder images in the `images/` folder:

| File | Description |
|------|-------------|
| `images/queen.jpg` | Main opening photo (appears under the crown) |
| `images/balloon1.jpg` | Photo revealed when balloon #1 is popped |
| `images/balloon2.jpg` | Photo revealed when balloon #2 is popped |
| `images/balloon3.jpg` | Photo revealed when balloon #3 is popped |
| `images/balloon4.jpg` | Photo revealed when balloon #4 is popped |
| `images/balloon5.jpg` | Photo revealed when balloon #5 is popped |
| `images/balloon6.jpg` | Photo revealed when balloon #6 is popped |
| `images/balloon7.jpg` | Photo revealed when balloon #7 is popped |
| `images/balloon8.jpg` | Photo revealed when balloon #8 is popped |

**Recommended image size:** 400x400 pixels (square images work best)
